zheatmap
========

This is a package that makes heatmap with dendrograms. To install the package, use the commands below:

``` r
devtools::install_github("zhuchcn/zheatmap")
```

Basic usage:

``` r
zheatmap(data)
```

See documentation:

``` r
?zheatmap
```
